CARACTERÍSTICAS NIVEL BÁSICO:

- Enunciados cortos (2-3 oraciones)

- Lenguaje simple y directo
- Incluir elementos visuales cuando sea apropiado

TIPOS DE EJERCICIOS MATEMÁTICAS:
- suma: Problemas de adición simple
- resta: Problemas de sustracción básica
- comparacion: Mayor/menor, más/menos
- conteo: Contar objetos, secuencias
- figuras: Identificar formas geométricas básicas
- patrones: Completar secuencias simples

PERSONALIZACIÓN POR ESTILO:
- Visual: Mencionar colores, formas, descripciones visuales

- Kinestesico: Incluir movimiento, acciones, actividades físicas
- Lectura_escritura: Enfocarse en leer y comprender el problema

PERSONALIZACIÓN POR CONTEXTO:
- mercado: Compras, frutas, verduras, precios
- escuela: Salón, útiles, compañeros, recreo
- hogar: Familia, comidas, juguetes, mascotas
- naturaleza: Animales, plantas, clima
- deportes: Juegos, equipos, puntos

INSTRUCCIONES ESPECÍFICAS:
1. Cada ejercicio debe tener exactamente 4 opciones (A, B, C, D)
2. Las opciones deben ser plausibles (no respuestas obviamente incorrectas)
3. Incluir distractor común (error típico que cometen los estudiantes)
4. La explicación debe ser clara y paso a paso
5. Usar nombres peruanos comunes
6. Adaptar dificultad al nivel_matematicas del perfil
